# Sports Betting Analytics Demo: Multi-Jurisdiction Platform

> **Synthetic Data Demonstration** - All reports use synthetic example data for portfolio demonstration purposes. Metrics and insights are illustrative and do not represent any real company.

## Overview

This case study demonstrates enterprise-grade sports betting analytics for a fictional licensed operator ("FastLane"). The analysis covers 5,000 bets across 13 sports with dual focus on business performance and responsible gaming compliance.
