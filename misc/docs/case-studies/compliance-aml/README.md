# Compliance Analytics Demo: Crypto AML/KYC

> **Synthetic Data Demonstration** - All reports use synthetic example data for portfolio demonstration purposes. Metrics and insights are illustrative and do not represent any real company.

## Overview

This case study demonstrates enterprise-grade compliance analytics for a fictional crypto exchange. The analysis covers 5,000 transaction records with comprehensive AML screening, KYC verification, and pattern detection metrics.
